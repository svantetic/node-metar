## Description

Get metar from VATSIM for given ICAO airport code and display it in terminal.

For simulation use only!

## Installation

```
$ npm install -g @svantetic/metar
```


## Usage

```
$ metar epkt
```

## Example output

```
EPKT 141430Z 09010KT 060V140 CAVOK 21/02 Q1015
```
